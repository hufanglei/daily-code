需求：1.自己编写代码生成一定数量的随机电话号码（注意：生成不同运营商的号码使用同一个方法）
		  2.对数据进行分区，规则：按照不同的运营商将电话号码分为三个区。
		  提示：移动电话号码以
		    134,135,136,
            137,138,139,
            150,151,152,
            157,158,159,
            188,187,182,
            183,184,178,
            147,172,198
			开头。
			电信电话号码以
			133,149,153,
            173,177,180,
            181,189,199
			联通电话号码以
			130,131,132,
            145,155,156,
            166,171,175,
            176,185,186,
			166
	手机电话号码都是11位，其中前三位是区分运营商，后4位是区分省地区，其余的是电话号码。
